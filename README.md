Foodmandu Website Clone

Project Overview:

This project is a "food ordering website" developed using HTML, CSS, and JavaScript.
It allows users to browse restaurants, view menus, and explore food items in a clean and user-friendly interface.


Objective:

The main objective of this project is to:

* Create a responsive food ordering website
* Improve UI/UX design skills
* Understand front-end web development concepts

Features:

1. Homepage

* Navigation bar with logo and icons
* Search bar to find restaurants or cuisines
* Featured restaurant cards with images and location

2. Restaurant Page

* Restaurant details (name, type, location)
* Menu categories (sidebar on desktop)
* Food items with price and description
* “Add to Bag” button

3. Mobile Responsive Design

* Sidebar converts into a dropdown menu on mobile
* Clean layout for small screens
* Improved user experience

4. Interactive Elements

* Clickable restaurant cards
* Add to cart button (UI-based)
* Dropdown menu selection


Technologies Used:

HTML – Structure of the website
CSS – Styling and layout
JavaScript – Interactivity

Design Highlights:

* Attractive color gradient design
* Food images for better user experience
* Clean and modern layout
* Better visual appeal compared to basic food ordering websites

Challenges Faced:

* Making the layout responsive
* Positioning elements correctly using CSS
* Designing mobile-friendly navigation
* Managing spacing and alignment

What I Learned:

* Responsive design using media queries
* UI/UX improvement techniques
* Using JavaScript for interactivity

Future Improvements:

* Add backend (PHP/MySQL)
* AI based recomendation system
* Real add-to-cart functionality
* Online payment feature
* Order tracking system

Conclusion:
This project demonstrates practical knowledge of front-end development and responsive design.
It can be further developed into a fully functional food delivery website.
